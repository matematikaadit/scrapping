Keterangan

Daftar calon penerima beasiswa MBKM 2013 ini diambil Website Kemahasiswaan UM. Didownload dan diolah dengan script (ruby & python) secara otomatis. Memanfaatkan beberapa library berikut: mechanize untuk scrapping, xlrd untuk parsing isi tabel di dalam excel, dan jinja2 untuk templating halaman. Filtering dan tema tabel diambil dari DataTables jquery plugin.

Project ini dalam rangka belajar pemrograman dengan memanfaatkan bahasa python (dan juga ruby). Harap maklum jika ada kesalahan baik dalam isi tabel ataupun halaman. Kritik dan saran alamatkan ke matematika dot adit at gmail dot com.

Kode untuk scrapping, parsing, dan templating serta halaman ini bisa dilihat di matematikaadit/scrapping.
